# 项目概述 python199

本项目是基于Python实现的二次开发音乐系统，通过django框架搭建后端服务，采用MySQL作为数据库支持，配备前端html+css+js技术栈，构建出一个功能丰富、易于扩展的音乐平台。

## 技术栈

- **前端**：HTML + CSS + JavaScript
- **后端**：Django
- **开发语言**：Python
- **数据库**：MySQL

## 功能模块

### 用户系统

- 用户登录
- 用户注册

### 用户界面

- 首页展示
- 音乐分类
- 新歌推荐

### 音乐详情页

- 听歌功能
- 歌曲下载
- 歌曲点评
- 歌曲信息展示
- 下一首推荐

### 搜索功能

- 音乐搜索

### 管理端

- 歌曲信息管理
- 歌曲分类管理
- 歌曲动态管理
- 歌曲评论管理
- 用户管理

## 系统角色

- 普通用户：可登录、注册、浏览音乐、下载音乐、评论音乐。
- 管理员：通过美化后的Django admin后台管理界面，上传音乐、管理歌曲信息、分类、评论以及用户。

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- 支持现代浏览器
